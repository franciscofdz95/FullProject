description: "The file describes the standard must be followed when a dropdown is required in an Angular application. It covers dropdown configuration, options, and data binding."
name: "Angular Dropdown Standard"
---

### Styles
- Use an angular dropdown component.
- Width must be 200px and height must be 40px..
- Font color of the header must be white.
- The dropdown must have a border of 1px solid #1D598E.
- The minimum width of the dropdown must be 150px.

### Formatting
- If the dropdown options include a date, it must be formatted as MM/DD/YYYY.

### Data Binding
- The dropdown must display a tooltip with the full value of the option when the value is truncated.

### Dropdown Properties
- Dropdown must be clearable.
- Dropdown must be searchable.
- Multiple selection must be supported.
